# TA Employment — Routine Task Checklist
**College of Arts, Media, and Technology (CAMT), Chiang Mai University**

> **Purpose:** A repeatable checklist for every semester/year cycle of the TA employment process.
> **How to use:** Duplicate this file for each semester (e.g., rename to `Routine Tasks 69-1.md`), then check off tasks as completed.

---

## 🔁 Cycle A — Annual Tasks
> Run **once per academic year**. Typically completed in **August–September**, submitted to Finance before October.

### Step 1 — Budget Planning (วางแผนงบประมาณ)

- [ ] **1.1** Collect draft timetable files (`.xlsx`) from Academic Service Staff (เจ้าหน้าที่สาขา) of each major
  - Files cover: Semester 2 (current year), Summer (current year), Semester 1 (next year)
  - Expected filename pattern: `(ร่าง) ตารางสอน [MAJOR] [YY]-[SEM].xlsx`
- [ ] **1.2** Retrieve historical TA assignment data from the past 3–4 academic years
- [ ] **1.3** List expected open courses at the **section level** for each of the three covered semesters
- [ ] **1.4** For each section, estimate TA positions (type, hours/week, weeks)
- [ ] **1.5** Calculate expected cost per position: `weeks × hours_per_week × hourly_rate`
- [ ] **1.6** Aggregate costs: section → course → major → grand total
- [ ] **1.7** Submit the completed budget plan to Finance Staff *(deadline: before October)*
- [ ] **1.8** *(Optional)* Remind เจ้าหน้าที่สาขา to include TA quota data (ยอดผู้ช่วยสอน) in future timetable files to reduce estimation work

## 🔁 Cycle B — Per-Semester Tasks
> Run **once per semester** (Semester 1, Semester 2, Summer).

---

### Step 2 — TA Quota Inquiry (สอบถามยอดผู้ช่วยสอน)
> Timing: ~**2 months before the semester starts**

- [ ] **2.1** Open EMS (`ems-admin.camt.cmu.ac.th`) and navigate to the current semester's TA position table
- [ ] **2.2** Enter each expected position from the Step 1 budget plan as a row in EMS
  - Columns to fill: Course Code, Section No., Program, Instructor, Class Type, Day(s), Start/End Time
  - For sections needing 2 TAs: use section suffixes `001-1` and `001-2`
- [ ] **2.3** Open EMS for instructors to review and select TA types
  - Instructors select TA type from the "โปรดเลือกประเภทผู้ช่วยสอน" dropdown
  - Instructors may add missing sections via "สร้างการขอใช้ผู้ช่วยสอน"
- [ ] **2.4** Verify that all sections have a TA type selected before closing this step
  - ⚠️ Note: "Graduate Assistant (International)" in EMS may represent either a Graduate Assistant or a Master Student Assistant — confirm with instructor if unclear
- [ ] **2.5** Finalize and close the quota inquiry phase

---

### Step 3 — TA Recruitment & Application (รับสมัครผู้ช่วยสอน)
> Timing: Open **at least 1 month before semester starts** (earlier is better)

**Phase 1 — Announcement**

- [ ] **3.1** Set recruitment dates:
  - Open date (`[OPEN_DATE]`)
  - Application deadline (`[DEADLINE_DATE]`, 23:59)
  - Results announcement date (`[RESULTS_DATE]`, 16:30)
- [ ] **3.2** Draft the bilingual Facebook post using the standard template (Thai + English)
- [ ] **3.3** Post announcement to the CAMT TA Facebook page (`https://www.facebook.com/camt.ta`)
- [ ] **3.4** Open the EMS application portal (`ems-admin.camt.cmu.ac.th/taExternal/login`) for applicants

**Phase 2 — Instructor Review (approx. 1 week after announcement)**

- [ ] **3.5** Notify instructors to log in to EMS and review applications for their sections
- [ ] **3.6** Monitor instructor responses (Accept / Reject / Move to another section)
- [ ] **3.7** Follow up with instructors who have not responded
- [ ] **3.8** Close the application portal and finalize selections on the results announcement date
- [ ] **3.9** Post results on Facebook (or notify via EMS as applicable)

---

### Step 4 — TA Appointment (แต่งตั้งผู้ช่วยสอน)

**Graduate Assistant (Thai & International Program)** — จ้างเหมา process

- [ ] **4.1** Prepare TOR (Terms of Reference) for each position
- [ ] **4.2** Prepare ใบเสนอราคา (Quotation) for each position
- [ ] **4.3** Collect and verify application documents from each selected TA (submitted in Step 3)
- [ ] **4.4** Submit จ้างเหมา documents to the relevant administrative office

**Master Student Assistant (International Program)** — Separate appointment process

- [ ] **4.5** *(Process details to be filled in)* — Follow the separate แต่งตั้ง process per university rules

**Bachelor Student Assistant, Lab Head, CAMT Staff**

- [ ] **4.6** *(Process details to be filled in)*

---

### Step 5 — Disbursement & Payment (เบิกจ่าย)
> Timing: At end of semester, after actual TA hours are confirmed

- [ ] **5.1** Collect actual weeks-assisted data for each TA per class assignment
- [ ] **5.2** Calculate actual payment per TA: `weeks_assisted × hours_per_week × hourly_rate`
- [ ] **5.3** Check against maximum wage cap: `15 weeks × hours_per_week × hourly_rate`
- [ ] **5.4** Flag any positions where actual wage exceeds the budget plan estimate ("เกินที่คาดการณ์")
- [ ] **5.5** Update actual wage (`ค่าจ้างจริง`) in EMS for each position
- [ ] **5.6** Prepare disbursement documents and submit to Finance Staff
- [ ] **5.7** Confirm payment processed for all TAs
