Budget Planning

วางแผนงบประมาณ
Once per year Aug–Sep · submit before Oct ▶

Estimate total expected TA cost for the three upcoming semesters (Sem 2 current year, Summer current year, Sem 1 next year), aligned with the Thai government fiscal year deadline.

Sub-steps

  • aCollect draft timetable files (.xlsx) from each major's Academic Service Staff.One file per major per semester; encourage them to include ยอดผู้ช่วยสอน (TA quota) data — done inconsistently, reminded each semester.
  • bTA Admin reviews historical TA assignment data (past 3–4 years).
  • cList expected opening courses at the section level for each covered semester.
  • dFor each section determine expected TA positions (type, hours, weeks).
  • eCalculate cost per position: weeks × hours/week × hourly rate; aggregate section → course → major → grand total.
  • fSubmit completed budget plan to Finance Staff before October.
Output: ▸ Budget plan — expected positions & costs per major + grand total

TA Quota Inquiry

สอบถามยอดผู้ช่วยสอน
Per semester ~2 months before start EMS ▶

Confirm which courses actually need a TA and how many, via EMS (ems-admin.camt.cmu.ac.th).

Phase 1 — Data input (TA Admin)

  • aEMSEnter each expected position from the budget plan as a row under the current semester.Multi-TA sections use suffixes — e.g. section 001-1, 001-2.
  • bOpen the system for instructors to review.

Phase 2 — TA type selection (Instructor)

⟳ Is the section already in the EMS table?
Yes → pick a TA type in the "โปรดเลือกประเภทผู้ช่วยสอน" dropdown.
No → click "สร้างการขอใช้ผู้ช่วยสอน" to create the section, then fill the full form.

Recruitment & Application

รับสมัครผู้ช่วยสอน
Per semester ≥ 1 month before start FBEMS ▶

Phase 1 — Announce & apply

  • aFBTA Admin posts the bilingual recruitment announcement (open / deadline / results dates).
  • bEMSApplicants apply at ems-admin.camt.cmu.ac.th/taExternal/login and upload required documents.Bachelor: app form, transcript, ID/passport, bank book. Grad/Lab Head/Master: CV, transcript, graduation/degree cert, ID/passport, bank book. (CAMT Staff exempt.)

Phase 2 — Instructor review (~1 week after post)

⟳ Instructor decision on each application
Accept → approve as TA for the position.
Reject → decline the applicant.
Move → reassign the application to another section.

Phase 3 — Results & report-in

  • aFBTA Admin posts selection results (bilingual, may be multiple rounds).
  • bFBSelected TA reports in: name, confirmed courses (CourseID Sec), status in semester; contacts the instructor for schedule/room.
  • cLINETA Admin replies — TA joins the semester Line group for payment-round updates (Line OA for private contact).
Output: ▸ Selected TAs confirmed & reported in

TA Appointment

แต่งตั้งผู้ช่วยสอน
Per semester Path differs by TA type ▶

The appointment route depends on the TA type group.

⟳ Which TA type?
Graduate Assistant (Thai & Intl) → employed via จ้างเหมา (contract) — needs: TOR (Terms of Reference), ใบเสนอราคา (quotation), and the application documents from Step 3.
Master Student Assistant (Intl) → separate appointment process (new university rule; distinct from the Graduate จ้างเหมา flow). Details TBD in KB.
Bachelor Student Assistant · Lab Head · CAMT Staff → appointment process to be documented in KB.
Output: ▸ Appointed TAs with employment documents in place
